# My Configurations

| Hotkey          | Function                |
| ----------------|:-----------------------:|
| Windows + Enter | Git Bash                |
| CapsLock        | [                       |
| Shift + CapsLock| (                       |
| Windows + Q     | Move Workspace Left     |
| Windows + W     | Move Workspace Right    |
| Windows + c     | Launch VS Code in Current Directory*    |

*Active window must be a directory in File Explorer
